# GuestQuest – Things to Do

A simple, responsive "Things to Do" listings page built with **HTML, CSS, JavaScript, jQuery, and Bootstrap 5** using local JSON data.

## Features
- Responsive card grid
- Live search & category filters
- Details modal with Swiper image slider
- Loading & empty states
- Request Info form with validation
- Interactive map using Leaflet & OpenStreetMap

## Run Locally
- Clone or download the project
- Open with **VS Code Live Server** or simply open `index.html`

## Live Demo
**https://guestquest-chi.vercel.app**
