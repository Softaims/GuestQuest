# GuestQuest – Things to Do Listings

A responsive, mobile-first "Things to Do" listings page built with HTML, CSS, JavaScript, jQuery, and Bootstrap 5 using local JSON data (no backend required).
Implemented responsive card grid, live search, category filtering, loading and empty states, and a detailed modal with a Swiper image carousel.
Added lazy-loaded images, a Request Info form with client-side validation, and a free interactive map inside the modal.
The application works across desktop, tablet, and mobile devices with a clean, modern UI.
To run, simply open `index.html` in a browser or use a local server (e.g., VS Code Live Server).
With more time, I would add filters, improved accessibility, animations, and API integration.
Development time: approximately 3–4 hours.
